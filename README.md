# Carbon + Next.js Starter

A starter template combining the power of [Next.js](https://nextjs.org) with IBM's [Carbon Design System](https://carbondesignsystem.com).

## Getting Started

To use this template:

1. Clone the repository
2. Install dependencies:

   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser to:
   ```
   http://localhost:3000
   ```

## Project Structure

- `app/` - App Router pages and layouts.
  - `globals.scss` - Global styles and Carbon theme imports.
  - `layout.tsx` - Root layout with Carbon styles integration.
  - `page.tsx` - Landing page component.
- `components/` - Reusable UI components.
  - `Header.tsx` - Global application header with navigation.
  - `ThemeToggle.tsx` - Component to handle theme switching.
- `public/` - Static assets.
